# 牌局计分

牌局计分上下文描述用户在一场牌局中加入、计分、结算和回看历史的核心语言。

## Language

**牌局**:
一组参与者在同一场结算周期内产生计分转移的活动。牌局在结束后形成可回看的历史记录。
_Avoid_: 游戏局, 房间, 会话

**牌局名称**:
用户为牌局输入的可读名称，并在展示时附加创建日期。牌局名称用于识别和回看，不作为唯一标识。
_Avoid_: 牌局 ID, 唯一名称, 房间名

**当前牌局**:
用户正在参与且尚未结束的唯一牌局。一个用户同一时间最多只能有一个当前牌局。
_Avoid_: 活跃房间, 当前游戏, 进行中会话

**邀请码**:
用于进入某个进行中牌局的短标识。邀请码在牌局结算或作废后失效。
_Avoid_: 牌局 ID, 二维码内容, 分享码

**加入确认**:
用户通过邀请码进入牌局前确认牌局信息的步骤。用户确认后才会成为参与者。
_Avoid_: 自动加入, 直接进房, 扫码即加入

**用户身份**:
进入小程序功能页后用于稳定识别同一个微信用户的身份。用户身份决定用户是否已有当前牌局，以及计分转移归属于谁；公开结算分享不要求用户身份。
_Avoid_: 昵称, 头像, 手机号

**展示资料**:
用户在牌局中展示给其他参与者看的昵称和头像。展示资料可以在牌局中修改，不改变用户身份或历史计分归属；昵称缺失时使用“老书记”加两位数字作为默认名称。
_Avoid_: 用户身份, 登录账号, 玩家 ID

**牌局展示资料**:
参与者在某个牌局内使用的昵称和头像。参与者修改牌局展示资料后，当前牌局、结算页面、历史详情和计分明细都按最新牌局展示资料显示；同一牌局内所有历史参与者的展示名称必须唯一。
_Avoid_: 用户身份, 登录账号, 全局资料

**参与者**:
已加入某个牌局、可产生或接收计分转移的用户。
_Avoid_: 玩家, 人员, 成员

**后加入参与者**:
牌局已经发生计分转移后加入的参与者。后加入参与者加入时最终分值从 0 开始，只参与加入后的计分转移。
_Avoid_: 补位玩家, 中途加入者, 新玩家

**人数上限**:
牌局允许同时存在的参与者数量，创建者计入人数上限。未设置人数上限的牌局不限制参与者数量。
_Avoid_: 玩家数量, 参与人数, 座位数

**牌局创建者**:
当前拥有牌局管理权的参与者。牌局创建者可以直接结束牌局，并负责确认其他参与者发起的结束请求；当前牌局创建者退出时，管理权转交给最早加入且仍在牌局中的当前参与者。
_Avoid_: 房主, 群主, 管理员

**退出牌局**:
当前最终分值为 0 的参与者离开尚未结算的牌局，并不再占用该牌局的人数上限。当前最终分值不为 0 的参与者不能退出，只能参与最终结算。
_Avoid_: 离开房间, 退局, 删除玩家

**退出参与者**:
已经退出牌局且最终分值为 0 的参与者。退出参与者不显示在当前参与者列表中，但会出现在结算页面和计分明细中。
_Avoid_: 删除参与者, 移除人员, 清空玩家

**历史参与者**:
曾经加入过某个牌局的参与者，包括当前参与者和退出参与者。结算页面显示所有历史参与者。
_Avoid_: 当前玩家, 有效成员, 结算人员

**恢复加入**:
退出参与者重新加入同一个尚未结算牌局。恢复加入不会创建新的参与者身份，参与者最终分值继续为 0。
_Avoid_: 重新注册, 新玩家加入, 重复加入

**作废牌局**:
尚未发生计分转移且最后一个当前参与者退出后的牌局。作废牌局不进入历史记录。
_Avoid_: 删除牌局, 取消房间, 解散

**结算牌局**:
已经结束并形成最终分值与计分明细的牌局。结算牌局进入历史记录。
_Avoid_: 完成牌局, 已结束房间, 结账

**最终分值**:
参与者在结算牌局中的累计结果。最终分值只由计分转移产生，所有参与者的最终分值总和必须为 0。
_Avoid_: 余额, 输赢金额, 总分

**历史牌局**:
当前用户曾作为参与者加入过的结算牌局。历史牌局可查看最终分值和全部计分明细。
_Avoid_: 全部牌局, 最近房间, 历史记录

**公开结算分享**:
结算牌局对外分享后的只读页面。公开结算分享允许任何获得分享入口的人查看，且不要求登录；公开结算分享只显示牌局名称、结算日期、历史参与者展示名称和最终分值，不显示头像或完整计分明细。
_Avoid_: 私密历史, 参与者历史, 继续加入链接

**计分明细**:
牌局中发生过的计分转移记录。多人计分转移在计分明细中可以合并显示为一条记录。
_Avoid_: 流水, 日志, 回合列表

**自动结算**:
系统在牌局长时间没有计分转移后，或有过计分转移且最后一个当前参与者退出后结束牌局。有过计分转移的牌局会自动成为结算牌局；从未发生计分转移的牌局会作废。
_Avoid_: 超时关闭, 自动结束, 定时解散

**计分转移**:
一个参与者把一定分值转给另一个或多个参与者；转出方扣分，接收方加分，牌局总分恒为 0。
_Avoid_: 轮次, 回合, 单方加分

**单人计分转移**:
一个参与者把一个分值转给另一个参与者。转出方扣除同等分值，接收方增加同等分值。
_Avoid_: 单独加分, 单人打分, 个人得分

**多人计分转移**:
一个参与者把同一个分值分别转给多个选定的当前参与者。转出方扣除该分值乘以接收方数量后的总分值，每个接收方增加该分值。
_Avoid_: 总额平分, 批量加总分, 全员平均分

**计分值**:
计分转移中每个接收方获得的正整数分值。计分值不允许为零或负数。
_Avoid_: 扣分值, 总额, 输赢差额

**结束请求**:
非牌局创建者的参与者请求结束牌局的意图。结束请求必须由牌局创建者确认后才会结束牌局。
_Avoid_: 申请解散, 结束投票, 结算申请
